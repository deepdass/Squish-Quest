# Squish Quest 
This is a 2D platformer game that follows a Knight on a quest defeating Slimes
<br>A short dummy project made to learn more about Godot

Made the game following the [tutorial](https://www.youtube.com/watch?v=LOhfqjmasi0) that everyone follows when first learning Godot and added Mario platformer mechanic

Learned about Godot and Basics of GDscript
